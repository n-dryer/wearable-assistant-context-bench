# Changelog

All notable benchmark-release changes live here.

## [v1] - 2026-04-21

Initial public v1 release of the **Wearable Assistant Context Benchmark**.

### Included

- README, benchmark spec, benchmark card, and benchmark notes as the
  public benchmark docs
- benchmark inputs and runner under `benchmark/v1/`
- frozen 11-scenario v1 set for reference-state selection under
  implicit context shift
- balanced Turn 2 accuracy under `baseline` as the primary ranking metric
- cross-family judge by default through `--judge-family auto`
- reproducibility manifest bundled with run outputs
