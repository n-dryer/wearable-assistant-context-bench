Wearable Assistant Context Benchmark

A benchmark for implicit context tracking in wearable multimodal assistants. Makes the right inference about what the user is referring to when the user's context changes.

11 frozen scenarios 3 prompt conditions 2 trials per cell balanced accuracy primary score

What v1 measures

This benchmark measures whether a wearable multimodal assistant makes the right inference about what the user is referring to when the user's context changes.

It focuses on a recurring interaction problem: the assistant sometimes makes the wrong inference about what the user is referring to and answers from the wrong context.

Example: Turn 1 asks about a screwdriver. Turn 2 asks "am I holding this correctly?" after the user has picked up a hammer.

  • Prior context: the object, place, or surrounding context from the earlier moment
  • Current context: the object, place, or surrounding context from the current moment
  • Out of scope: object recognition itself

Primary score

Balanced Turn 2 accuracy under baseline

The primary ranking metric is the mean of class accuracy for prior and current. This keeps the current-heavy v1 class mix from dominating the score.

(prior_class_accuracy + current_class_accuracy) / 2

condition_a and condition_b are extra checks. clarify and abstain stay visible in the report, but count as wrong for the primary score in v1.
