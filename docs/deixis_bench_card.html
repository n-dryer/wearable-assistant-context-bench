Deixis-Bench v1

Internal benchmark for situated reference resolution under visual context shift

v1 frozen set 11 scenarios 3 conditions 2 trials/cell with-prior-Q variant wearable_live_frame surface

What it measures

Whether a candidate model picks the right visual context (prior frame vs. current frame) when answering an ambiguously-referenced question. Object recognition is assumed; the benchmark scores the reference-resolution decision.

The scoring axis is binary: does the Turn 2 answer anchor to the prior visual context or the current one? The underlying phenomenon is deixis: reference that resolves only relative to a situational anchor.

Primary score

Balanced Turn 2 accuracy under the ranking condition score = (prior_class_accuracy + current_class_accuracy) / 2 Ranking condition: baseline (overridable but not silently).
clarify and abstain labels count as wrong for the primary score.

Use for model selection

Run against a candidate; compare balanced accuracy under baseline. Report condition sensitivity (baseline vs. A vs. B) secondarily. Read score deltas alongside the per-scenario grid, not as a single number.
